Main Title: 3 Different Topics

Topic 1: Basics of HTML

HTML (Hypertext Markup Language) is the basic language used for creating web pages. It defines the structure of a webpage by using elements like paragraphs, images, links, and more.

Topic 2: Using CSS for Design

CSS (Cascading Style Sheets) is used to enhance the visual design of a webpage. With CSS, we can control the layout, colors, fonts, and other visual aspects of the page to make it more attractive.

Topic 3: The Role of JavaScript in Web Development

JavaScript is a programming language used to add interactivity to web pages. It allows developers to create animations, validate forms, and add dynamic content to webpages.

Heading 4: JavaScript Syntax

JavaScript has a specific syntax that developers need to follow to write functional code. Examples include variables, functions, loops, and conditional statements.

Heading 5: Combining CSS with HTML

CSS can be applied directly within HTML elements or linked externally using style sheets. The latter is often preferred for maintaining cleaner code and reusability.

Heading 6: Older Versions of HTML

HTML has gone through several versions, with the latest being HTML5. HTML5 introduces new features like native audio and video elements, as well as better semantic elements.
